# iOS 단축어로 북마크 저장하기

이 문서는 iPhone/iPad 공유 시트에서 현재 페이지나 이미지를 `my-bookmark`에 저장하는 단축어 설정 방법입니다.

## 준비

1. 웹앱 → **설정 → API 키**로 이동합니다.
2. 이름을 입력하고 **발급**을 누릅니다.
3. 표시된 `bm_...` 키를 복사합니다.
   - 키 원문은 이 화면에서 한 번만 표시됩니다.
   - 키를 잃어버리면 회수 후 새로 발급하세요.
4. 서버 주소를 확인합니다. 예: `https://bookmark.example.com`.

> 스크린샷 자리: 설정 화면의 API 키 발급/복사 UI

## 레시피 A — AI 자동 분류로 저장

1. iOS **단축어** 앱을 엽니다.
2. **+** → 새 단축어를 만들고 이름을 `북마크 저장 (AI)`로 지정합니다.
3. 단축어 세부 설정에서 **공유 시트에서 보기**를 켭니다.
4. 공유 시트 입력 유형은 **URL** 또는 **Safari 웹페이지**를 허용합니다.
5. 액션 **URL 콘텐츠 가져오기**를 추가합니다.
6. URL을 다음으로 설정합니다.
   - `https://<서버주소>/api/bookmarks`
7. 방법을 **POST**로 설정합니다.
8. 헤더를 추가합니다.
   - `X-API-Key`: `bm_...`
   - `Content-Type`: `application/json`
9. 요청 본문을 **JSON**으로 설정하고 다음 값을 넣습니다.

```json
{
  "url": "단축어 입력",
  "mode": "ai"
}
```

10. 액션 **알림 표시**를 추가하고 `저장됨`이라고 표시합니다.
11. Safari에서 공유 버튼 → `북마크 저장 (AI)`를 실행합니다.

> 스크린샷 자리: URL 콘텐츠 가져오기 액션의 URL/헤더/JSON 본문 설정

## 레시피 A-이미지 — 공유한 이미지를 자동 분석해 저장

1. iOS **단축어** 앱에서 새 단축어를 만들고 이름을 `이미지 북마크 저장`으로 지정합니다.
2. 단축어 세부 설정에서 **공유 시트에서 보기**를 켜고 입력 유형은 **이미지**만 허용합니다.
3. 공유 시트 입력을 대상으로 **각 항목 반복** 액션을 추가합니다. 여러 장을 공유하면 한 장씩 독립된 항목으로 등록됩니다.
4. 반복문 안에 **URL 콘텐츠 가져오기** 액션을 추가합니다.
5. URL을 `https://<서버주소>/api/images`, 방법을 **POST**로 설정합니다.
6. 헤더에 `X-API-Key: bm_...`를 추가합니다. multipart 경계가 자동 생성되므로 `Content-Type` 헤더는 직접 넣지 않습니다.
7. 요청 본문을 **폼**으로 설정하고 필드 이름 `image`, 값은 현재 **반복 항목**으로 지정합니다.
8. 성공 횟수와 실패 횟수 변수를 각각 증가시키고, 반복이 끝나면 `성공 N장 · 실패 M장` 알림을 표시합니다.
9. 사진 앱에서 JPEG 한 장과 HEIC 한 장을 각각 공유해 등록되고 AI 분석이 시작되는지 확인합니다.

API 키 원문은 발급 직후 한 번만 표시됩니다. 키가 보이는 단축어 편집 화면이나 헤더 설정을 공유 스크린샷에 포함하지 마세요.

## 레시피 B — 카테고리를 직접 선택해 저장

1. 레시피 A의 1~4단계를 동일하게 진행하고 이름을 `북마크 저장 (선택)`으로 지정합니다.
2. 첫 번째 **URL 콘텐츠 가져오기** 액션을 추가합니다.
   - URL: `https://<서버주소>/api/categories`
   - 방법: **GET**
   - 헤더: `X-API-Key: bm_...`
3. 응답의 `items`에서 카테고리 이름(`name`)을 보여주고 값은 `id`가 되도록 **목록에서 선택** 액션을 구성합니다.
4. 두 번째 **URL 콘텐츠 가져오기** 액션을 추가합니다.
   - URL: `https://<서버주소>/api/bookmarks`
   - 방법: **POST**
   - 헤더: `X-API-Key: bm_...`, `Content-Type: application/json`
   - JSON 본문:

```json
{
  "url": "단축어 입력",
  "mode": "manual",
  "categoryId": "선택한 카테고리 id"
}
```

5. 액션 **알림 표시**를 추가합니다.

> 스크린샷 자리: 카테고리 목록 요청과 목록에서 선택 액션

## 미분류로 저장하기

레시피 A의 본문에서 `mode`만 `none`으로 바꾸면 됩니다.

```json
{
  "url": "단축어 입력",
  "mode": "none"
}
```

## 오류 처리 팁

- 이미 저장한 URL이면 API가 409를 반환합니다. 단축어에서 상태 코드가 409일 때 `이미 저장된 링크` 알림을 표시하도록 분기할 수 있습니다.
- 키를 회수하면 해당 키로는 401이 반환됩니다. 새 키를 발급하고 단축어 헤더를 교체하세요.
- API Key는 북마크/카테고리 API에서만 동작합니다. `/api/keys` 같은 키 관리 API는 웹 로그인(Bearer 토큰) 전용입니다.
